Python ile Matrix Efektinizi Yapın

Published 21/02/2022python12:29

Python ile matrix efekti yapımı hakkında bir video paylaşmıştım, buradan ulaşabilirsiniz. şimdi gelin basit bir şekilde python ile matrix efekti nasıl yapılır bunu görelim

Kütüphanelerimizi İmport Ediyoruz

Evet sadece 3 kütüphaneyi bu şekilde import ediyoruz, sizlerde aynı şekilde kütüphaneleri import edebilirsiniz

					    
import random
import time
import os

Kütüphane Hatası

Eger kütüphaneler ile ilgili bir hata alırsanız, onları yüklemeniz gerekecektir, hangi kod ide'sini kullanıyorsanız buna aşinasınızdır fakat mantık hep aynı, aşagıda verilen yükleme komutlarını çalıştırıp, kütüphaneleri yükleyebilirsiniz, eger sorun yaşamadıysanız buna gerek yok bir çok kütüphane zaten yüklü gelmektedir.

pip ile yükleyin:
  • pip install os
  • pip install random
  • pip install time
Kodlarımızı Yazalım
					    
Num = [0,1]
Tm = 0

Yukarıda Num değişkenin de bir liste oluşturduk, ve bu listede 0 ile 1 sayıları mevcut, daha sonra Tm değişkenine 0 verdik çünkü birazdan bir while döngüsü oluşturacağız, ve bu döngünün sonlanması için Tm değişkenini kullanacağız

Döngümüzü Oluşturalım
					    
while tm < 30:
    print(random.randrange(1,5)* " ",
    random.choice(nums),random.randrange(1,5)* " ",

Yukarıda bir döngü oluşturduk ve print komutunu kullanarak, Num listemizde bulunan 0,1 rakamlarını rastgele bir boşluk oluşturarak ekrana yazdırmasını söyledik, şimdi print içerisinde bunları virgulden itibaren kopyalayıp 35 adet oluşturalım
